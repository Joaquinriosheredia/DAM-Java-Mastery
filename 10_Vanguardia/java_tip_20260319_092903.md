## Técnica en Java: Creación de una Clase Simple

Para los estudiantes de DAM, aquí tienes un ejemplo breve y claro de cómo crear una clase simple en Java:

```java
// Se define la clase Persona
public class Persona {
    
    // Variables de instancia
    private String nombre;
    private int edad;

    // Constructor
    public Persona(String nombre, int edad) {
        this.nombre = nombre;
        this.edad = edad;
    }

    // Método get para obtener el nombre
    public String getNombre() {
        return nombre;
    }

    // Método set para establecer el nombre
    public void setNombre(String nombre) {
        this.nombre = nombre;
    }

    // Método get para obtener la edad
    public int getEdad() {
        return edad;
    }

    // Método set para establecer la edad
    public void setEdad(int edad) {
        this.edad = edad;
    }
}
```

Este código define una clase `Persona` con propiedades básicas y métodos getter/setter. Es un buen punto de partida para entender las estructuras de clases en Java.
