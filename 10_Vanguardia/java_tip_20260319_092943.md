# Técnica Breve: Uso del For-Each en Java

El `for-each` es una estructura de bucle simplificada que se utiliza comúnmente con colecciones. Su sintaxis es concisa y fácil de leer:

```java
// Ejemplo 1: Iteración sobre un array
String[] frutas = {"manzana", "banana", "cereza"};
for (String fruta : frutas) {
    System.out.println(fruta);
}

// Ejemplo 2: Iteración sobre una lista
List<String> colores = new ArrayList<>();
colores.add("rojo");
colores.add("verde");
colores.add("azul");

for (String color : colores) {
    System.out.println(color);
}
```

Este bucle `for-each` reduce la cantidad de código y mejora la legibilidad, ya que no requiere inicializar un contador ni manejar el índice explícitamente.
