#### 2. Arquitectura de la Solución

SpacetimeDB opera bajo una arquitectura que combina conceptos avanzados de bases de datos espaciales e informáticas para proporcionar rendimiento óptimo. Su diseño se basa en:

- **Espacio-Tiempo (Space-Time)**: Utiliza un modelo de datos multidimensional que integra coordenadas espaciales y temporales, permitiendo consultas complejas y rápidas.
  
- **Sistemas Distribuidos**: Implementa una arquitectura distribuida para mejorar la escalabilidad y reducir tiempos de respuesta.

- **Inteligencia Artificial (IA)**: Integra algoritmos de aprendizaje automático para optimizar consultas y predecir patrones de uso, mejorando aún más el rendimiento.

- **Consistencia Transacional**: Garantiza la consistencia transaccional a través del protocolo CAP (Disponibilidad, Consistencia, Partición soportada) al nivel del backend.

# Importar la librería de SpacetimeDB
from spacetime_db import SpacetimeDBClient

def consulta_spacetime():
    # Inicializar el cliente de SpacetimeDB
    client = SpacetimeDBClient('http://localhost:8080')

    # Definir los parámetros de la consulta spacio-temporal
    params = {
        "latitude": 37.7749,
        "longitude": -122.4194,
        "radius": 5000,  # en metros
        "time_range": (start_time, end_time),  # Intervalo de tiempo en segundos
    }

    # Ejecutar la consulta y obtener los resultados
    response = client.query_data(**params)

    for item in response:
        print(f"Ubicación: {item.location}, Datos: {item.data}")

# Ejecutar la función de consulta
consulta_spacetime()
