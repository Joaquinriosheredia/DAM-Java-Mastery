# Técnica Breve en Java: Uso de ArrayList

**Descripción:** `ArrayList` es una clase implementada en Java que permite almacenar colecciones dinámicas de objetos.

**Ejemplo:**

```java
import java.util.ArrayList;

public class EjemploArrayList {
    public static void main(String[] args) {
        // Crear un ArrayList vacío
        ArrayList<String> frutas = new ArrayList<>();

        // Agregar elementos al ArrayList
        frutas.add("Manzana");
        frutas.add("Banano");

        // Acceder a elementos
        System.out.println("Primera fruta: " + frutas.get(0));

        // Tamaño del ArrayList
        int size = frutas.size();
        System.out.println("Número de frutas: " + size);
    }
}
```

**Beneficios:** 
- Facilidad de uso para manejar colecciones dinámicas.
- Métodos útiles como `add`, `remove` y `size`.
```

Este breve ejemplo muestra cómo utilizar `ArrayList` en Java, incluyendo la creación del objeto, el añadido de elementos, y el acceso a los mismos.
