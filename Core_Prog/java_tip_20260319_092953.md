# Tip Técnico: Uso de la Clase `ArrayList` en Java

- **ArrayList** es una implementación de la interfaz `List` en Java.
- Utiliza `add(elemento)` para añadir elementos al final del ArrayList.
- Para acceder a un elemento, usa `get(indice)`.
- Para eliminar un elemento por su índice, usa `remove(indice)`.
- Puedes verificar si el ArrayList está vacío con `isEmpty()`.
- Para iterar sobre los elementos, utiliza un bucle for-each.

```java
ArrayList<String> lista = new ArrayList<>();
lista.add("Java");
lista.add("Python");

for (String l : lista) {
    System.out.println(l);
}
```
