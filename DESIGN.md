# 🎨 DESIGN.md - Google Stitch Standard (v1.0)

## 🎯 Global Vibe: "Executive Reliability"
- **Concept:** Una fusión entre la precisión quirúrgica (Sector Sanitario) y la robustez del código Java (DAM).
- **Visual Tone:** Dark Mode nativo, acentos en azul cobalto y tipografía sans-serif de alta legibilidad.
- **Core Principle:** Información crítica visible en < 2 segundos (Optimizado para emergencias).

## 🛠️ Mapping de Agentes (Stitch Specs)
- **UI Framework:** Android Jetpack Compose / JavaFX.
- **Consistency:** Todos los componentes generados por IA deben seguir el esquema de colores "Professional-Medical-Tech".
- **Accessibility:** Contraste mínimo 7:1 para entornos de baja visibilidad (Ambulancias/Logística nocturna).
