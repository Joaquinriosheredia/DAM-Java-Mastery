# Joaquín Ríos Heredia - Senior Java & SRE Portfolio

Este repositorio es el output de una **Factoría Automática de Activos Digitales** diseñada bajo estándares SRE (Site Reliability Engineering).

## 🚀 La "Bestia": Pipeline Local de Automatización
He implementado una infraestructura local (Ryzen 7 + Ollama 14b) que orquestra el ciclo de vida del desarrollo:
1.  **Inferencia Senior:** Soluciones basadas en patrones de diseño (SOLID, Clean Architecture).
2.  **Gatekeeping (CI):** Validación de compilación en tiempo real con **Maven** y **JDK 21**.
3.  **Auto-Documentación:** Generación dinámica de este índice maestro.

## 🛠️ Stack Tecnológico
- **Lenguajes:** Java 21/24, Python (Orquestador).
- **Backend:** Spring Boot 3.4, Resilience4j, HAPI FHIR.
- **IA Local:** Ollama (Qwen2.5 14b).
