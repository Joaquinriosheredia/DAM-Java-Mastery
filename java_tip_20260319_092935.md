## Tip Técnico: Uso de Arrays en Java

Arrays son estructuras de datos que almacenan elementos del mismo tipo. En Java, se pueden crear arrays utilizando la sintaxis `new` o declarando su tamaño explícitamente.

**Ejemplo 1:** Crear un array con `new`

```java
int[] numeros = new int[5]; // Crea un array de enteros de tamaño 5
```

**Ejemplo 2:** Inicializar y llenar un array

```java
String[] nombres = {"Juan", "María", "Carlos"}; // Inicializa el array directamente
for (int i = 0; i < nombres.length; i++) {
    System.out.println(nombres[i]);
}
```

**Ejemplo 3:** Uso de la clase `Arrays` para ordenar arrays

```java
import java.util.Arrays;

// ...

Arrays.sort(numeros); // Ordena el array 'numeros'
```
